Canvas: Mijn Digitale School


Beste ouders,


Wij gaan jullie vandaag laten zien hoe wij op school werken met Canvas. Canvas is een website waar alle spullen van school op staan. Het is eigenlijk ons hele lesprogramma in één applicatie.


Wat is Canvas?

Canvas is een website waar ik elke dag voor school opga. Hier staan alle vakken die ik heb. Elk vak heeft een eigen plek, dat heet een "tegel". In die tegel staat alles wat ik voor dat vak nodig heb.


Hoe gebruik ik Canvas?

Dit doe ik elke dag met Canvas:

1. Mijn vakken bekijken: Ik heb tegels voor alle vakken, zoals Nederlands, rekenen, programmeren en project.

2. Lezen en leren: In elke tegel staat de theorie. Dat zijn teksten, video's en uitleg van de docent.

3. Opdrachten maken: Alle opdrachten staan in Canvas. Ik kan ze hier maken en inleveren.

4. Mijn cijfers zien: Ik kan altijd kijken welke cijfers ik heb gehaald.

5. Kijken wat ik moet doen: Er is een kalender. Daar zie ik welke opdrachten ik deze week moet maken.

6. Berichten lezen: Mijn docenten sturen berichten via Canvas.


Waarom is Canvas handig?

- Alles voor één vak staat op één plek.

- Ik kan thuis ook bij mijn schoolspullen.

- Ik zie direct mijn cijfers.

- Ik vergeet geen opdrachten door de kalender.

- Mijn docenten kunnen mij helpen via berichten.


Wat ga ik jullie laten zien?

Ik laat jullie zien:

- Hoe mijn Canvas eruit ziet

- Welke vakken ik heb

- Hoe ik opdrachten maak

- Waar mijn cijfers staan

- Hoe de kalender werkt

- Hoe jullie met mij mee kunnen kijken en mij helpen met mijn planning
